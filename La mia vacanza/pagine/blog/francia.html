Ah la Francia...

Ah.. la Francia.. Prima di venire in Italia, vivevo in Francia con mia madre. Quanti ricordi.. Da quel tempo, mia madre è partita in un posticello fuori Parigi. Perché? Perché lì c'è di tutto. Proprio tutto. A meno di 5 minuti di camminata si può trovare: grande centro commerciale con tutti i tipi di negozi possibili, posta, negozio alimentare (Carrefour, grande negozio come Esselunga), una grande piazza dove tantissimi praticano il skateboarding, il municipio, un parco, un grande lago che si può percorrere in barca oppure dove si trova gente che pesca, oppure c'è chi fa il giro del lago correndo o camminando, una stazione di metro, tante stazioni di bus, ospedale, scuole, palestra, bar e ristoranti. Impressionante.


Ho stampato del denaro

Eh già. Grazie al funzionamento dei bitcoins, si può creare e stampare dei biglietti di denaro. Basta creare un nuovo indirizzo, e copiare il codice QR, che poi viene photoshopato su un biglietto personalizzato.


Sto creando una app

Ho imparato un nuovo linguaggio di programmazione, cioè Dart. È un linguaggio molto bello e giovane. Assomiglia molto a JavaScript, però è più focalizzato sulla programmazione a oggetti. Ad esempio, si può anche omettere il tipo delle variabili e delle funzioni, oppure si possono passare dei parametri per il loro nome. Ad esempio: int calcola(int num1, int num2) { return num1*num2; } int calcola2(int num1, int num2, int num3) { return num1*num2*num3; } Si potrebbe scrivere così in Dart: calcola(num1, num2, {num3}) { num3 != null ? return num1*num2*num3 : return num1*num2; } print(calcola(num3: 5, 4, 3)); È molto elegante e bello da scrivere e da vedere.
